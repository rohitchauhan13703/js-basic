JavaScript Basics

Foundation for Learning React

1. Variables (let, const)


// Using let (mutable)
let userName = "John";
userName = "Jane"; // Can be changed

// Using const (immutable)
const birthYear = 1990;
// birthYear = 2000; // Error: Can't reassign const

2. Data Types


// String
let greeting = "Hello World!";

// Number
let temperature = 23.5;

// Boolean
let isLoggedIn = true;

// Object
let user = { 
    name: "Alice", 
    age: 25 
};

// Array
let colors = ["red", "green", "blue"];

3. Functions


// Regular Function
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow Function
const addNumbers = (a, b) => a + b;

// Immediately Invoked Function Expression (IIFE)
(function() {
    console.log("This runs immediately");
})();

Start learning React after mastering these JavaScript fundamentals!
